Tested on endeavourOS and arch linux. 
Im making a really shitty themable music player in order to finally have one that fits with my linux rice :D
I neither know nor care about coding norms/etiquette so expect the most horrific, undebugable code ever.
It only plays mp3s btw

<p>Some Features:</p>
<p>-Auto-queue: Once youve already loaded a song any others you select are automatically added to your queue</p>
<p>-Customizability: You can change font color, background color and one accent color using hexcodes.</p>
<p>-Random Shuffle</p>
<p>-Cute ghost in the corner: Its just there.</p>
